
# VerticalAlignment
Enumeration of possible vertical alignment values.

## Enum
 Name | Description
------------ | ------------
**None** | No alignment (use specified position).
**Top** | Align to top.
**Center** | Center alignment.
**Bottom** | Align to bottom.
