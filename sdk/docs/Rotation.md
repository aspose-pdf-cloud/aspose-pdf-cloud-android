
# Rotation
Enumeration of possible rotation values.

## Enum
 Name | Description
------------ | ------------
**None** | Non-rotated.
**on90** | Rotated on 90 degrees clockwise.
**on180** | Rotated on 180 degrees.
**on270** | Rotated on 270 degrees clockwise.
