
# SoundEncoding
The encoding format for the sample data. 

## Enum
 Name | Description
------------ | ------------
**Raw** | Unspecified or unsigned values in the range 0 to 2^bits − 1.
**Signed** | Twos-complement values.
**MuLaw** | μ-law–encoded samples.
**ALaw** | A-law–encoded samples.
