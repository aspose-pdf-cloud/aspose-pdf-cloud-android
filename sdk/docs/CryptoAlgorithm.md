
# CryptoAlgorithm
Represent type of cryptographic algorithm that used in encryption/decryption
routines.

## Enum
 Name | Description
------------ | ------------
**RC4x40** | RC4 with key length 40.
**RC4x128** | RC4 with key length 128.
**AESx128** | AES with key length 128.
**AESx256** | AES with key length 256.
